# Proyecto Final - Data Science

Este proyecto analiza un conjunto de datos clínicos para predecir la presencia de enfermedad cardíaca utilizando técnicas de análisis exploratorio, selección de variables y modelos de clasificación.

## Contenido
- Análisis Exploratorio de Datos (EDA)
- Preparación de datos
- Feature Selection
- Modelo de clasificación (Regresión Logística)
- Evaluación con métricas
